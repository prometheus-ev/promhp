h2. <%= page_title %>

W&auml;hrend die meisten unserer Benutzer f&uuml;r ihre Arbeit mit dem Bildarchiv den Zugang &uuml;ber die Internetseite verwenden werden, besteht durchaus auch die M&ouml;glichkeit direkt mit "pandora":<%= pandora_url 'about' %>, Software von prometheus, dem digitalen Bildarchiv, zu sprechen. Wenn Sie nun aber einen Idee f&uuml;r eine interessante neue Anwendung haben - sei es Spiel, eine App f&uuml;r ein Mobiltelefon, eine Meta-Suchmachine, etc. - welche gewissermassen auf dem Bildarchiv aufsetzt, dann ist eine besondere Kommunikation mit dem System notwendig, &uuml;ber welche sich die ben&ouml;tigten Informationen abgreifen lassen. Dies ist der Punkt, an welchem die Bildarchiv API ins Spiel kommt!

Mit Hilfe der Schnittstelle ist es m&ouml;glich die folgenden Aktionen auszuf&uuml;hren:

* Suchen, 
* Abrufen von Bildern und
* Erkunden von Sammlungen.
