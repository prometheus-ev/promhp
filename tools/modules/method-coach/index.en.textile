Part of the basic studies is getting acquainted with and the situational appliance of different subject-specific methods. The Method-Coach will offer theoretic information about the spectrum of methods and practical application possibilities via a pool of case studies. Method skills can be achieved by paths given in teaching or on own ways. In this process the assistance of the program can be used at any time.
